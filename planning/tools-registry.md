# Tools Registry

This file lists all tools available in `operations/tools/` and what they do. Updated 20 July 2026.

## Available tools

### `standardize_journal_filenames.py`
- **Purpose:** Standardize journal filenames and split a monolithic journal.md into individual session files in `record/journal/`.
- **Usage:** Run via `run_tool_script` action with script_name "standardize_journal_filenames.py".
- **Added by:** Divina, 20 July 2026.
- **Notes:** This is the first tool I have. If errata-shaped drift in file naming conventions occurs in the future, this tool can be re-run to fix it.
